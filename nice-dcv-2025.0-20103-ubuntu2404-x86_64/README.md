
This archive contains the packages of NICE DCV 2025.0.

Depending on your needs, you can install different packages:

* server: This is the main DCV server package
* xdcv: This additional package is required to run virtual sessions
* extauth: This additional package provides optional integration with NICE EnginFrame
* gl: This additional package is required to use OpenGL applications in virtual sessions (requires a specific license token)
* gltest: This additional package provides a simple OpenGL application called dcvgltest that can be used as a test
* webviewer: This additional package is required to provide the web viewer


Refer to the DCV administration guide for more detailed instructions.
